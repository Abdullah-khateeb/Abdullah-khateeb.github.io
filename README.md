# Terminal Portfolio v2 🚀

A minimalist, high-performance terminal-themed portfolio designed for a DevSecOps and Cybersecurity professional. This version features a custom typing engine, Arch Linux aesthetic, and a modular architecture.

![Focus](https://img.shields.io/badge/Focus-Cybersecurity%20%7C%20DevSecOps-green)
![OS](https://img.shields.io/badge/OS-Arch%20Linux-blue)
![Shell](https://img.shields.io/badge/Shell-Bash-white)

## 🛠 Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox)
- **Logic:** Vanilla JavaScript (Asynchronous Typing Engine)
- **Font:** JetBrains Mono
- **Design:** Arch Linux Neofetch Aesthetic

## 📁 Project Structure

.
├── assets/             # Resume.pdf and other binary files
├── index.html          # Main structure & content
├── style.css           # Terminal UI & Dracula-inspired theme
├── script.js           # Typing animation logic
├── v1_backup/          # Archived legacy portfolio
└── README.md           # Project documentation

## 🚀 Key Features

- **Arch Linux Neofetch:** Custom ASCII art and simulated system status.
- **Dynamic Typing Engine:** Simulated CLI interaction for an authentic experience.
- **Responsive UI:** Fully optimized for mobile, tablet, and desktop viewing.
- **Verified Certifications:** Direct links to Credly, Cisco, and TryHackMe credentials.
